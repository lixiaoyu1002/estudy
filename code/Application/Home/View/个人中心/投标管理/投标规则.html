投标规则

所有竞标者均视同认同 ; 并愿意遵守及按照本拍卖下列各项竞标规则：

1. 本目录所有标价以新台币计算，如用外币以收款当日之台湾银行挂牌买入价之最低价折算，　未挂牌之外币恕不接受。

2. 每项以最高出标者为得标者，得标人须另付得标价之 10 ％为买方佣金。

3. 得标者之得标价为次高标价加一级最小递增额为得标价，但若该项无次高标时则以底价为得标价，买方佣金、邮汇费等费用另加。

4. 若二位或二位以上之得标价相同时，以先到者为得标人。 ( 以邮戳或传真时间为凭 )

5. 杸标后如需弃权，请于杸标截止前书面通知以利处理，并维护您的信用。

6. 本中心有权拒绝接受任何杸标，或收回任何所列之标售项目而不需解释。

7. 除非目录另有证明，请勿分拍或合拍任何项目，否则该项标单视为无效。

8. 得标之票品若发现与说明不符，可于收到得标品后七天内将原件退还，本拍卖将原款退还，而无须负担任何费用；但若得标之票品项目记载为下者，则不能退回。

a " 邮集 " " 杂票 " " 选集 " " 各项 " " 一批 " 等整批出售者 ;

b 原说明已载有瑕疵者 ;

c 记有存疑者 ;

d 现场取货者 ;

9. 若为有附图之项目则不能以齿孔或邮票图案或残缺为理由要求赔偿或退回。

10. 得标者于接获得标通知后，请于七日内将所有费用汇入本拍卖指定社账户，本中心接获后五日内将票品寄上，若经二次通知，仍不汇款取票，则往后之标单，本拍卖将予以拒绝。

11. 得标人之得标各项须另加佣金 10% 。

12. 投标价不得低于底价，标明估价者不得低于估价 70% 。
