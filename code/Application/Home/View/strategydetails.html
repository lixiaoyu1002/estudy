地铁小鲜肉17号线实现三线换乘，周边写字楼升值在即

所属分类
时间：2016-01-01
作者：宋媛

是否邻铁已经是写字楼选址的参考之一，本期空间观察为您梳理一下北京地铁的一条小鲜肉线路——计划于2019年全线运营通车的地铁17号线。

为什么要专访一条尚未通车的地铁线路？实在是因为它的气场太强大，17号线20座车站中有7座可与既有线路实现换乘，其中望京西站、十里河站、东大桥站、永安里站4座可换乘三条地铁线，即实现“三线换乘”，其中东大桥站换乘6号线和规划中的CBD线。

地铁17号线未来科技城北区站、南区站两座车站，以及邻近的三段隧道区间已在全线率先开工。17号线设车站20座，其中，换乘站共10座，达到一半。在这10座换乘站中，共有7座车站可与既有线路实现换乘。

最值得一提的是，17号线中的4座车站可实现“三线换乘”，即乘客在一个车站可换乘三条地铁线路，包括：

望京西站换乘13号线、15号线

北京香颂

北京香颂处于东北四环和京承高速夹角的60公顷绿地内，在季景沁园南侧，距离望京轻轨站300多米，总体建筑面积30余万平米，物业费：2.78元/平米•月。北京香颂规划为一个写字楼、公寓、精装公馆、商业、花园洋房和城市公寓多种物业类别多功能的集合式街区型产品。北京香颂依托由高速公路、城市干道、轨道交通、公交枢纽组成的立体交通网络，成为东北四环交通中心地。

十里河站换乘10号线和14号线

潘家园大厦

潘家园大厦位于朝阳区潘家园南里12号，出入二、三环路交通方便，大厦东临华威小区（东三环），西临东二环路，北临劲松小区，南侧步行8到达二环路，通过朝阳区政府规划部门了解，大厦南侧是北京市重点规划区，发展前景看好。物业费：6.00元/平米•月，建筑面积：28000平方米。

东大桥站换乘6号线和规划中的CBD线

蓝岛大厦

北京蓝岛大厦位于朝阳区朝外大街，毗邻使馆区和CBD中央商务区，门前有16条公交车经过，交通十分便利。

永安里站换乘1号线和规划中的R1线

中海广场

中海广场位于北京CBD核心地段，南面鸟瞰长安街，东侧毗邻国贸中心，占据国贸商务圈显要位置，由世界500强成员企业--中海地产根据国际甲级写字楼标准倾力打造。中海广场由两座国际甲级写字楼及北侧商业配楼和商业裙楼组成，总占地面积约1.9万平方米，建筑面积为15.1万平方米。
