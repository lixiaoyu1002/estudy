除非您已阅读并接受本规则所有条款，否则您无权申请或使用eStudy号码。您申请或使用eStudy号码的行为即视为您已阅读并同意受本规则的约束。
一、【规则的范围】

1.1 本规则是易学助手制定的关于获取和使用eStudy号码的相关规则。本规则适用于易学助手提供的需要注册或使用eStudy号码的全部软件和服务。
1.2 本规则属于易学助手的业务规则，是《易学助手服务协议》不可分割的组成部分。
1.3 您通过eStudy号码使用易学助手的软件和服务时，须同时遵守各项服务的单独协议。
二、【eStudy号码的性质】

eStudy号码是易学助手创设的用于识别用户身份的数字标识。eStudy号码的所有权属于易学助手。
三、【eStudy号码的获取】

3.1 您可以通过如下方式免费或有偿申请注册eStudy号码，包括但不限于：
（1）软件客户端；
（2）易学助手网站；
（3）手机短信、电话语音（IVR）；
（4）参与易学助手开展或授权开展的活动；
（5）其他易学助手授权的方式。
3.2 您完成申请注册程序后，依照易学助手的业务规则获得该号码的使用权。
3.3 无论您通过何种方式获取eStudy号码，均受本规则约束。
四、【用户身份信息验证】

4.1 您在申请eStudy号码过程中，需要填写一些必要的信息，请保持这些信息的及时更新，以便易学助手向您提供帮助，或更好地为您提供服务。若国家法律法规有特殊规定的，您需要填写真实的身份信息。若您填写的信息不完整或不准确，则可能无法使用服务或在使用过程中受到限制。
4.2 易学助手与用户一同致力于个人信息的保护，保护用户个人信息是易学助手的一项基本原则。未经您的同意，易学助手不会向易学助手以外的任何公司、组织和个人披露您的个人信息，但法律法规另有规定的除外。
五、【eStudy号码的使用】

5.1 您可以按照易学助手的业务规则用eStudy号码登录和使用易学助手提供的各种软件和服务。
5.2 除了易学助手提供的软件和服务，您还可以用eStudy号码登录使用易学助手合作伙伴的相关服务。
六、【eStudy号码的安全】

6.1 eStudy密码由您自行设定。您应妥善保管您的eStudy号码与密码。
6.2 易学助手与您共同负有维护eStudy号码安全的责任。易学助手会采取并不断更新技术措施，努力保护您的eStudy号码在服务器端的安全。
6.3 您需要采取特定措施保护您的号码安全，包括但不限于妥善保管eStudy号码与密码、安装防病毒木马软件、定期更改密码等措施。
七、【eStudy号码的找回】

如果您的eStudy号码被盗、密码遗忘或因其他原因导致无法正常登录，您可以按照易学助手的申诉途径进行申诉，并遵守《eStudy帐号申诉服务协议》。易学助手并不承诺您一定能通过申诉找回eStudy号码。
八、【用户行为规范】

8.1 您不得恶意注册eStudy号码。易学助手可以对恶意注册行为进行独立判断和处理。
8.2 eStudy号码使用权仅属于初始申请注册人。未经易学助手许可，您不得赠与、借用、租用、转让或售卖eStudy号码或者以其他方式许可非初始申请注册人使用eStudy号码。
8.3 如果您当前使用的eStudy号码并不是您初始申请注册的或者通过易学助手提供的其他途径获得的，但您却知悉该eStudy号码当前的密码，您不得用该号码登录或进行任何操作，并请您在第一时间通知易学助手或者该号码的初始申请注册人。

9.1 您理解并同意，作为eStudy号码的初始申请注册人和使用人，您应承担该号码项下所有活动产生的全部责任。
